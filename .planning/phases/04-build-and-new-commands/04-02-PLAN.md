**Step 2: Create `src/commands/new/parse-input.ts`** with these exports:

```typescript
export function parseInputName(input: string): { slug: string; name: string }
```
Split ONLY on `[-_\s]` (NOT on `/`). Slug = `parts.join('-')`. Name = PascalCase (`parts.map(p => p[0].toUpperCase() + p.slice(1)).join('')`). Guard against empty parts.

```typescript
export function inferTypeAndName(args: string[]): { typeArg: string | undefined; nameArg: string | undefined }
```
Takes process.argv. Slice(2) to get args after 'node qwik'. Filter out flags (starting with `--`). Get mainInput from non-flag args after the command name ('new'). Apply rules:
- Starts with `/` and ends with `.md` -> markdown type, strip `.md` from name
- Starts with `/` and ends with `.mdx` -> mdx type, strip `.mdx` from name
- Starts with `/` -> route type
- Non-empty without `/` -> component type
- Empty -> undefined/undefined

```typescript
export function inferTemplate(argv: string[], hasPositional: boolean): string | undefined
```
Extract flag args (starting with `--`), strip `--` prefix. If any flag arg found, use it as templateId. If hasPositional is true and no flag, default to `'qwik'`. Otherwise undefined (interactive).

**Types:**
```typescript
export type TemplateType = "component" | "route" | "markdown" | "mdx";

export interface TemplateFile {
  filename: string;  // e.g. "index.tsx.template" or "[slug].tsx.template"
  content: string;
}

export interface Template {
  id: string;        // e.g. "qwik"
  component?: TemplateFile[];
  route?: TemplateFile[];
  markdown?: TemplateFile[];
  mdx?: TemplateFile[];
}
```

**loadTemplates():**
- Resolve stubs/templates/ directory using `import.meta.url`:
  ```typescript
  const __filename = fileURLToPath(import.meta.url);
  const templatesDir = join(dirname(__filename), '..', '..', '..', 'stubs', 'templates');
  ```
  This navigates from `src/commands/new/templates.ts` -> up 3 levels to project root -> `stubs/templates/`.
- Read subdirectories of templatesDir (each is a template ID, e.g. "qwik")
- Sort with 'qwik' first
- For each template ID, read each type subdirectory (component, route, markdown, mdx)
- Read all files in each type dir (exclude `.gitkeep`), returning filename + content
- Return `Template[]`

**writeTemplateFile(outDir, templateFile, slug, name):**
- `mkdirSync(outDir, { recursive: true })` -- NEW-07
- Compute output filename: replace `[slug]` with slug, strip `.template` extension
- Compute full output path: `join(outDir, outFilename)`
- **Duplicate guard (NEW-04):** If `existsSync(fileOutput)`, throw `new Error('"${outFilename}" already exists in "${outDir}"')` -- uses outFilename (the final filename), not the template filename
- Replace `[slug]` and `[name]` tokens in content
- `writeFileSync(fileOutput, content, 'utf-8')`

**getOutDir(rootDir, typeArg, nameArg):**
- For `component`: return `join(rootDir, 'src', 'components')` -- FLAT, no nameArg subdirectory (matches test NEW-02 assertion of `src/components/counter.tsx`)
- For `route`, `markdown`, `mdx`: return `join(rootDir, 'src', 'routes', nameArg)` -- nameArg includes leading `/` which path.join normalizes

Export all functions and types.
