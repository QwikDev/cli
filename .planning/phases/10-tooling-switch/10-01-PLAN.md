   ```bash
   pnpm add -D vite-plus
   pnpm remove @biomejs/biome
   ```

2. Delete `biome.json` (the file at repo root).

3. Delete `vitest.config.ts` (superseded by vite.config.ts -- never have both at the same level per research pitfall 3).

4. Create `vite.config.ts` at repo root with this exact content:
   ```typescript
   import { defineConfig } from "vite-plus";

   export default defineConfig({
     lint: {
       ignorePatterns: ["dist/**", "node_modules/**"],
       rules: {
         "no-unused-vars": "error",
         "no-console": "off",
       },
     },
     fmt: {
       ignorePatterns: ["dist/**", "node_modules/**"],
       tabWidth: 2,
       useTabs: false,
       semi: true,
       singleQuote: false,
       trailingComma: "all",
     },
     test: {
       include: ["tests/unit/upgrade/**/*.spec.ts"],
     },
   });
   ```

   Key settings rationale:
   - `singleQuote: false` matches Biome's existing double-quote behavior (minimize reformatting diff)
   - `trailingComma: "all"` matches Biome's default
   - `tabWidth: 2` + `useTabs: false` matches biome.json `indentWidth: 2` + `indentStyle: "space"`
   - `test.include` scoped to ONLY `tests/unit/upgrade/**/*.spec.ts` -- preserves Japa/Vitest separation (research pitfall 2)

5. Update `package.json` scripts section to:
   ```json
   {
     "build": "tsdown",
     "test": "node --import tsx/esm bin/test.ts",
     "test:unit": "vp test",
     "lint": "vp lint",
     "lint:fix": "vp lint --fix",
     "format": "vp fmt",
     "format:check": "vp check"
   }
   ```
   Keep all other package.json fields unchanged.

Do NOT modify `bin/test.ts` -- its Japa exclusion pattern (`!tests/unit/upgrade/**`) is already correct and unchanged.
